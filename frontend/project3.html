Project Description

This project is a full-stack portfolio web application that showcases projects, interactive UI components, and responsive design. Users can navigate through different sections, view project details, and experience an intuitive interface.

Key Features

  • Responsive design for mobile and desktop
  • Interactive UI components (cards, modals, buttons)
  • Dynamic project display with detailed views
  • Clean, modern color palette and typography

Technologies Used

HTML, CSS, Bootstrap, JavaScript, (optional: Node.js / Express for backend)
